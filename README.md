## What is it?
Symbiose is an associative website designed to address one of the Sustainable Development Goals (SDGs): No Poverty.
It contains two sub-projects: a video game and a short film. Each addressing an additional SDG: Gender Equality and Reduced Inequalities (Disability).

#### The project was created by a team of 7 people:

- 1 developer
- 2 marketing & communication specialists
- 3 designers

It is mainly built using HTML, CSS, and JavaScript as part of a school project.
